ТЕГИ

Тег параграф p для вывода текста

Тег ul указывает на маркированный список

Тег li определяет пункт списка

Тег ol создаёт нумерованный список

Для выделения текста жирным используется один из двух парных тегов:bили strong

Тег i предназначен для визуального выделение текста курсивом и не несёт смысловой нагрузки

Тег em акцент на эту часть предложения курсивом

АТРИБУТЫ

Для появления всплывающей подсказки использовался атрибутtitle

ЗАГОЛОВКИ

Для создания заголовков используется 6 парных тегов: h1, h2, h3, h4, h5, h6

ТАБЛИЦЫ

table — область таблицы, tr — строка, thead — шапка таблицы, th — ячейка шапки, tbody — тело таблицы, td — ячейка
